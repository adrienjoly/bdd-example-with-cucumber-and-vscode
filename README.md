I'm discovering BDD (Behavior Driven Development) with Benjamin Hugot, using Cucumber/Gherkin and JavaScript.

How to run the tests?

```sh
$ nvm use # to use the version of node.js specified in .nvmrc
$ npm install
$ npm test
```

You should get:

```
..

1 scenario (1 passed)
2 steps (2 passed)
```
